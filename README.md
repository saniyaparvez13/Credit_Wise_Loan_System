# Credit_Wise_Loan_System

Built an end-to-end machine learning classification model using Python and Scikit-learn to predict loan approval status based on applicant financial and demographic data. Performed data preprocessing including handling missing values, feature engineering, and encoding categorical variables. Conducted exploratory data analysis (EDA) to understand data distributions and relationships, along with correlation heatmap analysis to identify feature dependencies. Applied train-test split to prepare data for model training and evaluation. Trained classification models and evaluated performance using accuracy, precision, and recall metrics to ensure reliable and balanced predictions. The system improves decision consistency, reduces manual bias, and enhances efficiency in loan approval processes.
